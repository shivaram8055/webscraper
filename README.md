# webscraper
This is an application which can gather mobiles from flipkart website just by copying the url of that website.
WebScrapper will create a csv file which makes us to differentiate each mobile easy.
These mobiles are differentiated with rating, price, reviews from the flipkart website.


#HOW TO RUN:
1.Run the main.py file.
2.Copy the link from the flipkart of which mobiles you want to differntiate.
3.Now choose a name for the .csv file to be saved.
4.Checkout the .csv that is created after the code is successfully complied and run.
